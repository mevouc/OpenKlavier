# OpenKlavier

## Overview

OpenKlavier is a desktop piano application, written in C#/.NET. Strongly inspired by [OpenPiano](https://github.com/Justagwas/OpenPiano)

## Requirements

- Windows
- .NET 10.0+

## Building

```bash
cd OpenKlavier
dotnet build
```

## Running

```bash
cd OpenKlavier
dotnet run
```
